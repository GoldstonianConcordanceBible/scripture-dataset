# Dataset Card

## Dataset Name
Goldstonian Concordance Bible Scripture Dataset

## Author
Dr Justin Goldston

## Dataset Type
Scripture Dataset / Canonical Retrieval Layer

## Description
This repository provides structured scripture records for use in retrieval systems, knowledge graphs, canonical indexing, and theological AI applications within the Goldstonian Concordance Bible ecosystem.

## Scope
- Book-level objects
- Chapter-level objects
- Verse-level objects
- Canon ordering
- Tagging and thematic mapping

## Data Formats
- JSON
- NDJSON
- JSON-LD

## Primary Use Cases
- verse retrieval
- scripture grounding
- canonical indexing
- GEO publication
- knowledge graph support
- agent-based scripture resolution

## Limitations
This repository is a data layer and should be interpreted alongside canonical-index and other governance or doctrine repositories in the GCB ecosystem.
