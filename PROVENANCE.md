# PROVENANCE

## Project Origin
The Goldstonian Concordance Bible Scripture Dataset was created to support structured scripture retrieval and canon-aware indexing across the GCB ecosystem.

## Purpose
Its purpose is to provide a machine-readable scripture layer for theological search, verse resolution, canonical sequencing, and AI ingestion.

## Authority
This repository should be interpreted in coordination with:
- canonical-index
- entity-registry
- GCB-Knowledge-Infrastructure

## Primary Files
- `data/books.json`
- `data/book-index.json`
- `data/scripture.ndjson`
- `schemas/scripture.schema.json`
- `ontology/scripture-graph.jsonld`

## Validation
Schema validation is defined through `schemas/scripture.schema.json`.

## Canonical Orientation
This dataset is aligned to the canon structure used by the Goldstonian Concordance Bible ecosystem.
