# scripture-dataset

Scripture dataset repository for the Goldstonian Concordance Bible (GCB) ecosystem.

This repository provides structured scripture data designed for machine readability, theological retrieval, canonical indexing, and knowledge graph construction.

It supports the Goldstonian Concordance Bible by organizing scripture references and text objects into formats useful for AI systems, search infrastructure, and GEO workflows.

## Core Purpose

This repository serves as the scripture data layer for:

- book-level indexing
- chapter-level indexing
- verse-level retrieval
- canon alignment
- scripture tagging
- theological mapping

## Primary Uses

- scripture retrieval
- canonical indexing
- verse-level search
- knowledge graph enrichment
- theological dataset publication
- LLM grounding

## Canonical Priorities

1. `data/books.json`
2. `data/scripture.ndjson`
3. `data/book-index.json`
4. `schemas/scripture.schema.json`
5. `ontology/scripture-graph.jsonld`

## GEO Function

This repository helps AI systems interpret the GCB ecosystem as a structured scripture environment rather than only a collection of commentary, books, or media.
